Docker Swarm Untuk Pemula

Mulai dari mode Swarm sampai aplikasi kamu live sebagai service

Halaman ini adalah jalur cepat untuk belajar orkestrasi container. Ikuti 6 langkah berurutan, jalankan command-nya, lalu deploy service pertama kamu di Docker Swarm.

Mulai Langkah Buka Versi Detail
01

Cek Docker Aktif

Pastikan Docker daemon berjalan normal di mesin manager.

docker version
02

Inisialisasi Mode Swarm

Aktifkan Swarm pada node pertama agar menjadi manager.

docker swarm init
03

Verifikasi Node

Lihat daftar node dan status role manager/worker.

docker node ls
04

Build Image Aplikasi

Buat image dari project sebelum dijadikan service.

docker build -t nodejs-app:1.0 .
05

Deploy Service

Jalankan aplikasi sebagai service dengan 3 replika.

docker service create --name webapp --replicas 3 -p 3000:3000 nodejs-app:1.0
06

Monitor dan Scale

Cek status task lalu tambah replika sesuai kebutuhan trafik.

docker service ls docker service ps webapp docker service scale webapp=5

Command Cleanup

Kalau ingin berhenti, hapus service dulu lalu tinggalkan mode swarm.

docker service rm webapp docker swarm leave --force
