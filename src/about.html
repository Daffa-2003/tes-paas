Tentang Proyek Ini

Proyek demonstrasi sederhana menggunakan Node.js dan Docker

Tujuan Proyek

Proyek ini dibuat untuk menunjukkan cara membuat aplikasi Node.js sederhana dan mengemasnya menggunakan Docker.

Dengan proyek ini, Anda dapat belajar:

  • Membuat aplikasi web sederhana dengan Node.js dan Express
  • Mengemas aplikasi Node.js dalam Docker container
  • Membuat Dockerfile yang efisien
  • Menjalankan aplikasi dalam lingkungan yang terkonsistensi dengan Docker

Teknologi yang Digunakan

Node.js

Runtime JavaScript untuk backend

Express.js

Framework web untuk Node.js

Docker

Platform containerization

HTML5 & CSS3

Untuk frontend yang responsif

Docker Commands

Build Image

docker build -t nodejs-app .

Membuat Docker image dari Dockerfile

Run Container

docker run -p 3000:3000 nodejs-app

Menjalankan container dari image yang dibuat

Run in Background

docker run -p 3000:3000 -d nodejs-app

Menjalankan container di background (detached mode)

Lihat Container

docker ps

Melihat container yang sedang berjalan
